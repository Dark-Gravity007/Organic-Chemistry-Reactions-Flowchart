## üåü Overview

Navigating organic synthesis can be overwhelming. This tool serves as a "GPS" for chemical reactions, helping students and educators visualize how different functional groups interconnect.

The application is a **single-file web solution**‚Äîno installation required. Simply open the HTML file in any browser to start exploring the networks.

## üó∫Ô∏è What's Inside?

The flowcharts are divided into four primary networks:

1.  **Alkyl Halide Network (R‚ÄìX):** The central hub for nucleophilic substitution, Grignard reagents, and name reactions like Finkelstein and Swarts.
    
2.  **Carbonyl Group (‚â•C=O):** Detailed pathways for Aldehydes and Ketones, including Reduction (Clemmensen, Wolff-Kishner), Nucleophilic Addition, and condensation reactions (Aldol, Cannizzaro).
    
3.  **Benzene & Phenol:** Electrophilic Aromatic Substitution (EAS) routes, side-chain reactions of Toluene, and specific Phenol tests.
    
4.  **Aniline & Diazonium Salts:** The "Bridge of Organic Chemistry"‚Äîshowing how to transform Aniline into various aromatic compounds via Diazonium intermediates.
    

## ‚ú® Key Features
